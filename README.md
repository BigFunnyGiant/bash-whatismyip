# WhatIsMyIp
This is a whatismyip bash script for linux.
Depends on a bunch of services, picks one randomly and executes curl to determine your wan address. It returns only that, nothing else.

# Usage
whatismyip.sh

# Dependencies
- Bash
- Curl
